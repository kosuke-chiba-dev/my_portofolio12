# DMM WEBCAMPでデザインしたLPをコーディング
仮想のパン屋を想定し、ターゲットとなるペルソナを設定しました。

そして、ムードボードの作成やコンテンツ構造の設計を経て、LPをデザインし、実際にコーディングしました。

さらに、バナー広告もデザインしました。

レスポンシブ対応済みです。
